## 使用

### 安装

`npm install -g management-cli`

### 初始化项目

- `vkdmc config set company companyName`
- `vkdmc init`

## 运行

### 安装依赖

`npm install`

### 启动

`npm run watch`

### 执行 `npm link`

此时就可以使用 `vkdmc` 命令了。

- `vkdmc config set company companyName`
- `vkdmc init`
- `vkdmc config get`

### 发布

开发完成后，即可发布至 npm.
